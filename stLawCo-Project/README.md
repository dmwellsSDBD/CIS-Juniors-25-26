# St. Lawrence County Landing Page - Complete Lesson Package

A comprehensive 90-minute lesson for 11th Grade Computer Science students focusing on responsive web design using Flexbox, CSS Grid, and modern HTML/CSS techniques.

## üìö Project Overview

Students create a responsive landing page for St. Lawrence County, New York, incorporating local context from the North Country region. The project teaches fundamental responsive design principles through a practical, real-world application.

### Learning Objectives
- Master Flexbox for navigation layouts
- Implement responsive CSS Grid systems
- Create hero sections with overlaid content
- Build functional contact forms
- Apply mobile-first responsive design methodology

### Key Features
- ‚úÖ Flexbox navigation bar (sticky positioning)
- ‚úÖ Two responsive hero sections with call-to-action buttons
- ‚úÖ CSS Grid card layout (1/2/3 columns based on screen size)
- ‚úÖ Footer with CSS Grid layout and contact form
- ‚úÖ St. Lawrence County color scheme (#003366, #0066CC, #FFB81C)
- ‚úÖ Mobile-first responsive design approach

## üìù Common Issues & Quick Fixes

### Images Not Displaying
- **Check:** Image URL in browser first
- **Try:** `https://picsum.photos/1200/600` as placeholder

### Grid Not Responsive
- **Check:** Media query syntax `@media screen and (min-width: 768px)`
- **Check:** `grid-template-columns` changes at each breakpoint

### Navigation Stacking Vertically
- **Check:** `.navbar .container` has `display: flex`
- **Check:** `.nav-menu` has `flex-direction: row`

### Hero Text Not Visible
- **Add:** White color and text-shadow
- **Try:** Darker hero image

### Hero Content Not Centered
- **Check:** Parent has `position: relative`
- **Check:** Content has `position: absolute` and `transform: translate(-50%, -50%)`
